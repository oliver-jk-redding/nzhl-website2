# nzhl-website

Community website for New Zealand Hobbit League

## how

### install

```
git clone https://github.com/oliver.jk.redding/nzhl-website
cd
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
